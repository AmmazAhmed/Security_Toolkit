#  Security_Toolkit 

An educational cybersecurity toolkit with authentication, encrypted messaging, password management, file encryption, phishing awareness, and classical ciphers.

## Features
- **User Authentication** - Register/Login with SHA-256 hashing
- **Secure Messaging** - AES-encrypted chat
- **Password Manager** - Encrypted vault with password generator
- **File Encryption** - Encrypt any file with password
- **Phishing Simulator** - URL analyzer + dynamic quiz
- **Cipher Toolkit** - Caesar cipher, Beaufort (1-40 key length), AES (auto key generater)
- 👑 **Admin Panel** - 
View all users and their data
